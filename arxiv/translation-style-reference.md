# Эталон стиля arXiv-переводов

Эталонная страница: `2606.15834.html`. Для всех HTML-переводов в `veai_vika/arxiv` действуют правила:

- Hero содержит ровно `<span class="eyebrow">технический перевод</span>`.
- В `<h1>` остаётся только содержательный русский заголовок: без arXiv ID, без английского названия и без служебных кластеров.
- Описание `.orig` краткое и не повторяет полное название статьи, если оно уже есть в заголовке.
- CTA в hero: одна ghost-кнопка Veai с уникальным продуктовым текстом страницы; кнопки `Читать` нет.
- Ссылки в основном тексте, `.orig` и `.card-doc` голубые, подчёркнутые и не жирные (`font-weight:400`).
- Подписи к изображениям начинаются с `Рисунок N.`; ссылки в тексте на рисунки оформляются как `(рис.N)`.
- Англоязычные термины переводятся или получают русскую расшифровку в скобках при первом важном употреблении.
- Длинное тире не используется; вместо него короткое `-`.
- Footer единый для всех страниц: продуктовые ссылки, соцсети, контакты и блок компании со ссылкой `https://veai.ru/about`.
- В данном дайджесте в footer и тексте не использовать международный бренд и связки с ним; бренд только `Veai`, почта и контакты только Veai.
- Копирайт в footer: `© Veai. Материал подготовлен по данным arXiv.org.`.
- Уникальные заголовки, текст, изображения и продуктовые CTA каждой статьи сохраняются.
- Язык: пишем простым литературным русским, полными предложениями, без англицизмов-калек и метафор.
- Запрещённые слова (не использовать никогда): «вслепую», «гадание»/«гадать», «инспекции» (пишем «статический анализ»), «Грунтинг»/«grounding».
